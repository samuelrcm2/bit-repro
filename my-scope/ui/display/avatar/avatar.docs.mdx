import Avatar from './avatar';

This component may be used to build page header contaning user information.

## Avatar's basic usage

```jsx
<Avatar />
```

## Avatar properties

The Avatar can be customized changing its content, its shape and background color

```jsx
<Avatar text="JF" color="red" shape="square" />
```
