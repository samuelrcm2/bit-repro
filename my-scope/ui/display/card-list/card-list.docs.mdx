import CardList from './card-list';

The CardList component may be used to render specific content as an profile or graphs from a dashboard.

## CardList's basic usage

This component can be used to render a List of cards utilizing the renderItem and dataSource property.

```jsx
<CardList renderItem={renderItem} dataSource={dataSource} />
```
