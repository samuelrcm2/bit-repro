import Image from './image';

The Image component may be used to render images in other components such as Avatar and Form.

## Image's basic usage

```jsx
<Image src="link-to-your-image" />
```

## Image properties

The width, height and description for the image can be defined.

```jsx
<Image width={200} height={300} alt="Image description" />
```
