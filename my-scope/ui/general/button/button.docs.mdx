import Button from './button';

This component may be used to build buttons in any size/color and text.

## button's basic usage

```jsx
<Button title="Simple Button" onClick={() => console.log('Clicked')} />
```

## Button properties

The Button has a variety of pre-defined buttons, as Cancel and Delete

```jsx
<Button.Cancel styles={{color="red"}}  />
```
