import Text from './text';

The text component can be used to render a text texts

## Text's basic usage

```jsx
<Text title={'Hello I`m text'} />
```

## Customizing properties

A type can be added to your text (danger,success,error)

```jsx
<Text title={'Hello I`m text in a danger type'} type="danger" />
```

A size can be added to your text to change the size

```jsx
<Text title={'Hello I`m text in a danger type'} type="danger" size="code-02" />
```
