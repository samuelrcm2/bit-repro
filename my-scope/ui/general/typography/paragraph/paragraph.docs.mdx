import Paragraph from './paragraph';

The paragraph component can be used to render a paragraph texts

## Paragraph's basic usage

```jsx
<Paragraph title={'Hello I`m paragraph'} />
```

## Customizing properties

A type can be added to your paragraph (danger,success,error)

```jsx
<Paragraph title={'Hello I`m paragraph in a danger type'} type="danger" />
```

A size can be added to your paragraph to change the size

```jsx
<Paragraph
  title={'Hello I`m paragraph in a danger type'}
  type="danger"
  size="code-02"
/>
```
