import Title from './title';

The title component can be used to render a title texts

## Title's basic usage

```jsx
<Title title={'Hello I`m title'} />
```

## Customizing properties

A type can be added to your title (danger,success,error)

```jsx
<Title title={'Hello I`m title in a danger type'} type="danger" />
```

A size can be added to your title to change the size

```jsx
<Title
  title={'Hello I`m title in a danger type'}
  type="danger"
  size="code-02"
/>
```
