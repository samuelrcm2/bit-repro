## Icon Component

Renders an icon on the screen. The icons are provided by the MUI library. To check available icons names, please go to [Material Icons](https://mui.com/material-ui/material-icons/?theme=Outlined) website.

### Component basic usage

```js
<Icon name="Home" />
```

### Using props to customize the icon

Modify the name to see the icon change live:

```js live
<Icon name="Home" />
```
